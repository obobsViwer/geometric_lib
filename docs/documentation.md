# **Документация по проекту** 


## 1. Общее описание решения

 Проект представляет собой библиотеку для вычисления **значений геометрических фигур**.  
 Были созданы функции для вычисления периметра и площади круга и квадрата.  
 Библиотека реализована на языке программирования **Python**. Также проект разрабатывался с помощью **системы контроля версий Git**.
  
 #### Проект включает в себя: исходные файлы, историю git и документацию в формате Markdown. 


## 2. Описание каждой функции с примерами вызова


###  **Файл circle.py**

#### **Функция area(r);**
Вычисляет площадь круга.

**Пример:**
```
from circle import area
print(area(3))  
    
result 28.274333882308138
```  

###  **Файл square.py**

#### **Функция perimeter(a);**
Вычисляет периметр квадрата.

**Пример:**
```
from square import perimeter
print(perimeter(5))
    
result 20
```
